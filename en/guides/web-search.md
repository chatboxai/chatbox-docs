# Web Search Mode

You can configure the web search service provider in "Extensions".

* Chatbox: A solution for Chatbox AI subscribers. Do not select this option when using other model service providers;
* Bing: Use Bing as the information source for web-connected Q&A;
* Tavily: Requires configuration at Tavily, enter your Key to use.
